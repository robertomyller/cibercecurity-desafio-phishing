Desafio de Cibersegurança: Phishing com Kali Linux.

 Descrição
Projeto desenvolvido como parte do desafio de cibersegurança da DIO. O objetivo é simular um ataque de phishing utilizando ferramentas do Kali Linux.

 Objetivos
- Compreender os fundamentos de engenharia social
- Criar uma página falsa para coleta de credenciais
- Utilizar Kali Linux para simular o ataque

 Tecnologias Utilizadas
- Kali Linux
- Apache Server
- Social Engineering Toolkit (SET)
- Git e GitHub

Capturas de Tela
As imagens estão organizadas na pasta `/images`.

 Recursos

- [Repositório Base](https://github.com/cassiano-dio/cibersecurity-desafio-phishing)

 Como Executar
1. Clone o repositório
2. Inicie o Apache Server
3. Execute o SET
4. Acesse a página falsa via IP local
